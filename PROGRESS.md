Kernel is the classic Chapman 2013 / LearnOpenGL hemisphere: each sample = unit-sphere direction with z≥0, multiplied by a uniform [0,1] length, then rescaled by a quadratic `0.1 + 0.9*(i/N)^2` falloff (early samples cluster near origin for contact AO; later samples reach wider). `std::mt19937(seed)` so CPU mirror and GPU kernel UBO upload (RP.5b) stay bit-identical for a given seed. Single shader + pipeline; two dispatches per frame with ping-ponged descriptor sets (H: `direction={1,0}` + ao target A→B, then V: `direction={0,1}` + ao target B→A). Shape mirrors `SsaoPipeline` (owns own layout+pipeline, deleted copy/move, `Bind(cmd)` at `VK_PIPELINE_BIND_POINT_COMPUTE`, ctor-failure layout cleanup). Descriptor set at set 0: `{0 = AO input (combined-image-sampler R8), 1 = linear depth pyramid (combined-image-sampler) for edge gating, 2 = R8 AO output (storage image)}`. Push constants: 12 bytes (`ivec2 direction, float edgeSharpness`) — tiny block so the two back-to-back dispatches only re-push the direction. Shader runs in 8×8 tiles, 7-tap symmetric gaussian (center + 3 pairs; weights `{0.265, 0.212, 0.108, 0.035}` sum to 1 pre-rejection). Edge rejection: per-tap weight is `baseWeight * exp2(-|Δd|/max(d, 0.01) * edgeSharpness)` — normalising by centre depth makes `edgeSharpness` a scale-invariant tuning knob, ε-guarded against the sky-sentinel `far` marker so the blur doesn't divide-by-near-zero over cleared pixels. Post-rejection weights are renormalised by the actual accumulated sum (crisp edges at the cost of `max(sum, 1e-4)` on pathologically-rejected pixels).
